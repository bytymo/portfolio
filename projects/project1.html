Local Library

Link to page Link to GitHub

Description:

A dummy website to mimic a library's administrative database. The HTML and CSS were already provided, requiring only the JavaScript be developed. The completed project provided a home page for general statistics regarding rented books and popular categories. The "Stats by Book" page provided a list of every book housed by the library, with a history of recent borrowers, the author, genre, and if is is currently returned or rented out. Finally, the "Stats by Account" page listed every library member alphabetically by last name, with a profile picture, general information about the member, as well as how many total books have been borrowed by the member.
